# Trabalho N1 Estrutura de Dados
Trabalho de Simulação de Spool de Impressão em Java
Feito por Kaue e Manoel

# 📌 Como Usar
Baixe ou clone o repositório:
git clone https://github.com/seu-repositorio/spool-impressao-java.git
Abra o projeto em uma IDE Java (Eclipse, IntelliJ, NetBeans ou VS Code).
Execute a classe principal do sistema (ex: Main.java ou Sistema.java).

# 📚 Funções do Sistema
Login inicial
->Cadastro de clientes
->Cadastro de produtos
->Criar pedidos
->Gerenciar fila de impressão
->Exibir pedidos e itens cadastrados
