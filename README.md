# Trabalho N1 Estrutura de Dados
Trabalho de Simulação de Spool de Impressão em Java<br>

Feito por Kaue e Manoel<br>

# 📌 Como Usar
Baixe ou clone o repositório:
```bash
git clone https://github.com/KaueOliveira0/EstruturaDeDados.git
```
Abra o projeto em uma IDE Java (NetBeans ou VS Code).<br>
Execute a classe principal do sistema kaueManoel.java<br>

# 📚 Funções do Sistema
->1 - Pedido de Impressão<br>
->2 - Iniciar a Impressão<br>
->3 - Finalizar a Impressão<br>
->4 - Mostrar a fila de Impressão <br>
->5 - Mostrar Total de Impressão<br>
->6 - Cancelar o Proximo pedido de Impressão<br>
->7 - Mostrar o Documento atual que esta sendo impresso<br>
->0 - Sair do Sistema<br>
